All UI designs must follow the tech-blue color theme based on the reference image at `public/img/banner-bg.jpg` (Lễ Kick-off — Hội thi Chuyển đổi số Ngành Ngân hàng 2026), but not use the image itself as backgrounds.

**Color palette (extracted from the banner):**
- Deep navy (base / corners): `#020b1e`, `#041436`
- Primary blue (mid gradient): `#0a4aad`, `#1660c5`
- Electric blue (mid-bright): `#1e90ff`, `#3a7bd5`
- Bright cyan accent (rays, highlights, focus glow): `#4dc9f6`, `#00bfff`
- Soft cyan highlight (ambient glow, thin lines): `#7dd3fc`, `#a3e2ff`
- Text on dark: white (`#ffffff`), or light blue-tinted white (`#e0f2fe`) for subtitles
- Subtle elements: blue-tinted grays (e.g. `rgba(0, 191, 255, 0.15)` borders)

**Style characteristics:**
- Tech / futuristic feel with neon glow effects
- Radial or diagonal gradient backgrounds: deep navy → electric blue → soft cyan halo
- Glowing thin lines, holographic circles, faint digital grid patterns
- Clean bold white typography, often uppercase with generous letter-spacing
- Cyber / digital aesthetic with bright rays emanating from focal points
- Avoid flat solid colors — prefer layered gradients + soft glows

**Why:** User explicitly requested this as the standard theme for all future UI work in this project (Chuyển đổi số Ngành Ngân hàng 2026 — Banking Digital Transformation 2026). The banner is the visual north star for the whole competition.

**How to apply:** When creating or styling any UI component, page, or admin interface, use this color palette and style. Default to dark navy backgrounds with bright cyan/blue accents and white text. Reach for `#4dc9f6` / `#00bfff` for the primary interactive accent, `#1e90ff` / `#0a4aad` for secondary surfaces, and `#041436` / `#020b1e` for base backgrounds.
