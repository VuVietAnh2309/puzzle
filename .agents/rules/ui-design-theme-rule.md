All UI designs must follow the tech-blue color theme based on the reference image at `public/img/banner-bg.jpg`, but not use the image at all.

**Color palette:**
- Background: dark navy to blue gradient (`#0a1628` → `#0d3b8f`)
- Primary accent: bright blue / cyan (`#00bfff`, `#4dc9f6`)
- Secondary accent: electric blue glow (`#1e90ff`, `#3a7bd5`)
- Text: white (`#ffffff`) on dark backgrounds
- Subtle elements: blue-tinted grays, neon blue lines

**Style characteristics:**
- Tech/futuristic feel with neon glow effects
- Gradient backgrounds (dark navy to medium blue)
- Glowing lines, holographic circles, digital grid patterns
- Clean white typography on dark blue backgrounds
- Cyber/digital aesthetic

**Why:** User explicitly requested this as the standard theme for all future UI work in this project (Chuyen doi so nganh ngan hang 2026 - Banking Digital Transformation 2026).

**How to apply:** When creating or styling any UI component, page, or admin interface, use this color palette and style. Default to dark blue backgrounds with bright cyan/blue accents and white text.
